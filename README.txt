README for File Splitting Tool
This executable program allows you to split large .csv, .tsv, or Excel files into smaller chunks. Follow these steps to use the tool:

1. Double-click the executable file to run the program.
2. When prompted, select the file you want to split.
3. Enter the number of splits you want for the file.
4. Optionally, choose a numeric column for summation before splitting.
5. Select the destination folder to save the split files and choose whether to open it afterward.
